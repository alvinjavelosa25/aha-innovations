Step 1: Upload JavaScript File

First, upload the ghl-auto-google-signin.js file to your web server or CDN so it's accessible via URL.

Example URL: https://your-domain.com/ghl-auto-google-signin.js

Step 2: Add JavaScript to GHL

Step 3: Add CSS to GHL (Optional)

In your GHL account, go to Settings ‚Üí White Label ‚Üí Custom CSS and add the contents of ghl-login-auto-signin.css for enhanced styling.

Step 4: Update Your Domain

In the JavaScript file, replace https://your-domain.com/ with your actual domain where the script is hosted.

üîß How It Works

The auto-signin script triggers when:

  • URL contains parameters: auto_google=true, from_signup=true, or google_redirect=true
  • User came from your signup domain (localhost:8080 or aha-innovations.com)
  • Email field is pre-filled (indicates redirect from your system)

üéØ Testing

Test the Auto-Signin

  1. Go to your signup page: http://localhost:8080/signup
  2. Click "Continue with Google"
  3. Use an email that already has a GHL account
  4. You should see "Found your account!" message
  5. You'll be redirected to GHL with auto-signin triggered
‚ö†Ô∏è Important: Make sure your GHL login page has a Google sign-in button visible. The script looks for various button selectors and will attempt to click the first one it finds.

üêõ Troubleshooting

If Auto-Signin Doesn't Work

  1. Open browser developer tools (F12)
  2. Check the Console tab for error messages
  3. Look for messages starting with "GHL Auto Google Sign-In Script"
  4. Verify the script is loading correctly
  5. Check if the Google button selector needs updating
