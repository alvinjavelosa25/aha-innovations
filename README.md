# AHA Innovations

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Features

- **Homepage**: Hero section with features preview and call-to-action
- **Features Page**: Detailed feature cards with benefits and integrations
- **Pricing Page**: Three pricing tiers with FAQ section
- **Sign In/Sign Up Pages**: Authentication forms with social login options
- **Server-Side Rendering**: All pages use Next.js App Router with automatic SSR
- **Responsive Design**: Mobile-friendly UI built with Tailwind CSS
- **Dark Mode Support**: System preference-based dark mode

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js App Router pages and layouts
- `/components` - Shared React components (Navigation, Footer)
- `/public` - Static assets

## Build for Production

```bash
npm run build
npm start
```
